Trabalhando com seletores

Id's

O id é um elemento que serve para identificar um item especifico. Abaixo veremos as suas caracteristicas

  • Ele só pode ser usado uma unica vez por
  • Pode ser usado junto a uma tag class
  • Sua hierarquia sobrepõe as demais (class e o identificador generico)
  • Para se referir a um id dentro do doc CSS basta por #nomeDoId
  • Podemos também usar elemento#nomeDoId

Class

Já a tag class é usada para identificar itens no geral dentro de um documento html. Se liga nessas caracteristicas:

  • Ela pode ser usada varias vezes dentro de um doc
  • Podemos chamar mais de uma classe dentro da mesma tag
  • Para se referir a uma classe dentro do doc CSS basta por .nomeDaClasse
