Trabalhando com fontes

Caracteristicas da tag css

Italico

A forma italica se da pelo uso do paramentro font-style: italic

Peso

O peso da fonte se da pelo parametro font-weight: ... tendo n variações como:

  • normal
  • bold
  • 100
  • 200
  • ...
  • 900

Tamanho

Partindo agora para o tamanho das fontes usamos o parametro font-size: ... para definir ele.
Geralmente usamos as medidas em "px" ou "em"(Unidade de medida de relação ao tamanho padrão) para passar o tamanho.
